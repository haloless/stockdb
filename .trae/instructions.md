# Project Instructions for StockDB

## Overview
StockDB is a Python/Flask application for analyzing China stock market data.

## Development Guidelines
- Follow PEP8 style guide
- Use descriptive variable names
- Write docstrings for all public functions

## Project Structure
- `stockdb2/` - Core application code
- `static/` - Frontend assets
- `tests/` - Test files
- `dbapp2.py` - Flask application entry point

## Key Workflows
### Running the Application
```bash
python3 dbapp2.py
```

### Running Tests
```bash
python3 -m unittest discover
```
